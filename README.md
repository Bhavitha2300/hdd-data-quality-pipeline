# Harmonized Drug Dataset Data Analysis Project

## Project Goal

The goal of this project is to learn and demonstrate a complete data analysis workflow using the Harmonized Drug Dataset (HDD). The project focuses on data ingestion, data quality checks, data cleaning, SQL analysis, and visualization using Spotfire.

## Tools Used

- VS Code
- Python
- MySQL
- Git and GitHub
- Spotfire

## Dataset

The Harmonized Drug Dataset integrates compound metadata, bioassay information, binding data, toxicity data, adverse drug reaction data, and molecular fingerprints.

## Current Project Phases

1. Project setup
2. Raw data inspection
3. MySQL ingestion
4. Data quality checks
5. Data cleaning
6. SQL analysis
7. Spotfire visualization
8. Final reporting
